## Problema vs Algoritmo

Un problema computacional define qué salidas son correctas para cada entrada.
Un algoritmo es un procedimiento que garantiza producir una salida correcta
para cualquier entrada válida del problema.


## Correctitud de Algoritmos

Para entradas arbitrariamente grandes, la correctitud se demuestra usando inducción.
Si el algoritmo funciona para k elementos y sigue funcionando para k+1,
entonces funciona para cualquier tamaño de entrada.


## Complejidad Temporal

La eficiencia de un algoritmo se mide por el número de operaciones
en función del tamaño de entrada n, usando notación asintótica.
